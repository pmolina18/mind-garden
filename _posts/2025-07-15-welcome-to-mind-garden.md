## What is Mind Garden?

Mind Garden is a personal blog for publishing thoughts on diverse topics without a rigid thematic structure. It is a space to explore ideas, share knowledge, and grow as a thinker.

## Why Start a Blog?

There are many reasons to maintain a personal blog:

- **Clarify your thinking** — writing forces you to organize your thoughts
- **Share knowledge** — help others learn from your experiences
- **Build in public** — document your journey and progress
- **Create a reference** — your future self will thank you

## The Tech Stack

This blog is built with a simple but effective stack:

```yaml
# _config.yml excerpt
title: "Mind Garden"
url: "https://pmolina18.github.io"
baseurl: "/mind-garden"
markdown: kramdown
plugins:
  - jekyll-feed
  - jekyll-seo-tag
```

The key components include:

1. **Jekyll** for static site generation
2. **GitHub Pages** for hosting
3. **GitHub Actions** for automated deployment
4. **Markdown** for content authoring

## Getting Started with Posts

Creating a new post is straightforward. Add a Markdown file to the `_posts/` directory following the naming convention:

```
YYYY-MM-DD-your-post-title.md
```

Each post needs front matter at the top with metadata like the title, date, and tags. Then write your content using GitHub Flavored Markdown — headings, lists, code blocks, links, and more are all supported.
